RUN THESE COMMANDS IN TERMINAL:

git clone https://github.com/retrospecs0801/civic-project.git |
cd civic-project


Backend setup (myproject folder):
cd myproject |
python -m venv venv |
venv\Scripts\activate  |
pip install -r requirements.txt |


Frontend setup (civic reporter): |
cd ../civic-reporter |
npm install |
npm run dev |
